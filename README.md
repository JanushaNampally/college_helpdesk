# College Helpdesk Chatbot

##Project Description
This project is a web-based college helpdesk chatbot developed using Django. 
It helps students get instant answers to common college-related queries like admissions, fees, timetable, and courses.

## eatures
- Student query response system
- Admin panel to add chatbot responses
- Simple web interface
- AI-based intent matching (basic)

## Technologies Used
- Python
- Django
- HTML, CSS
- SQLite Database
